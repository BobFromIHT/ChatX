# ChatX for ATLYSS by Bob from IT

- Chat Opacity Slider.
- Toggle a tall chat window to expose more message history.
- Toggle 24h `[HH:MM]` or 12h `[hh:MM tt]` timestamps on messages.
- Toggle `*asterisk italics*` for emphasis.
- Toggle name underline and ping notification when someone else mentions you.
- Toggle colored `(G)/(P)/(R)` prefixes so you always know which channel a message came from, even on custom themes.
- Toggle routing gameplay server messages to the Game feed.
- Enable OOC messages with `//` or toggle OOC for the session with `/ooc`.
- Click links in chat to copy them or open them in the default browser.
- Chat character counter.
- Block all Chat messages.
- Block all Game Feed messages.
- Clear Chat and Game Feed keybind.
- Toggle scrollbar visibility.
- Toggle Chat visibility keybind.
- Bump the message limit from 125 to 500 characters.
  *(Must be enabled on both client and server.)*

---

## Settings

All options are exposed as friendly toggles and sliders in Nessie's EasySettings panel.

---

## Requirements

- Nessie's EasySettings
- BepInEx
